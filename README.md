# 📊 Telecom Customer Churn Analysis Dashboard using Power BI

## 📌 Project Overview

This project presents an interactive **Power BI dashboard** developed to analyze customer churn in the telecom industry. The dashboard provides valuable insights into customer behavior, contract types, payment methods, internet services, and demographic factors that influence customer churn. The objective is to help businesses identify at-risk customers and improve customer retention strategies through data-driven decision-making.

---

## 🎯 Objectives

- Analyze customer churn trends.
- Identify factors contributing to customer churn.
- Compare churn across different customer segments.
- Monitor important business KPIs.
- Support customer retention strategies using data visualization.

## 📊 Key Insights

- Customers with **Month-to-Month contracts** have the highest churn rate.
- Customers with **shorter tenure** are more likely to leave.
- **Fiber Optic** users show comparatively higher churn.
- Customers using **Electronic Check** as their payment method have higher churn.
- Customers with **One-Year** and **Two-Year contracts** have significantly lower churn.
- Long-term customers are more likely to remain with the company.
